# Aula 01 - Fundamentos de Banco de Dados

## Objetivos

Ao final desta aula você será capaz de:

- Entender o que é um banco de dados.
- Diferenciar dados e informação.
- Identificar tabelas, registros e campos.
- Compreender a importância dos bancos de dados nos sistemas atuais.

---

# O que é um Banco de Dados?

Um banco de dados é um local organizado para armazenar informações.

Praticamente todos os sistemas utilizam bancos de dados:

- Instagram
- Netflix
- Spotify
- WhatsApp
- Sistemas Escolares

---

# Analogia: Biblioteca

Imagine uma biblioteca.

| Biblioteca | Banco de Dados |
|------------|---------------|
| Estante | Tabela |
| Livro | Registro |
| Informação do livro | Campo |
| Biblioteca inteira | Banco de Dados |

---

# Dados x Informação

## Dados

São valores isolados.

Exemplos:

- João
- 18
- Blumenau

## Informação

Dados organizados.

| Nome | Idade | Cidade |
|--------|--------|---------|
| João | 18 | Blumenau |

Agora temos informação.

# Estrutura Básica

## Tabela

Tabela ALUNO

| id | nome | idade |
|----|------|--------|
| 1 | Ana | 17 |
| 2 | João | 18 |

---

# Campos

São as colunas.

```text
id
nome
idade
```

---

# Registros

São as linhas.

```text
1 | Ana | 17
```
